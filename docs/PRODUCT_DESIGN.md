# Antigravity-Go（AGo）v0.2.1 产品设计

## 1. 产品定义
Antigravity-Go（AGo）是一个基于 Go 开发的工业级智能体运行环境与本地工作台。系统通过 `antigravity_core` 提供感知基础设施，由宿主程序完成任务编排、闭环执行、结果验证和系统自愈。

## 2. 核心目标
- **稳定宿主**：提供可控的内核生命周期管理、运行时隔离与异常恢复能力。
- **闭环执行**：打通感知、决策、执行、验证与回滚的完整工程链路。
- **工程交付**：为命令行、Web 工作台与部署场景提供一致的产品化入口。

## 3. 核心能力

### 3.1 内核宿主层
负责内核全生命周期管理：
- 启动编排与元数据注入（标准输入 TLV 格式）
- 端口自动发现（基于日志模式匹配）
- 状态自检（`WaitReady` 与端口探测）

### 3.2 智能体运行时
实现基于外部大模型的决策循环：
- **工具调用**：连接本地工具与内核 RPC 能力
- **权限管理**：提供显式审批策略
- **上下文管理**：支持摘要、裁剪与记忆写回

### 3.3 交互层
- **TUI**：基于 Bubble Tea 的命令行对话界面
- **Web 工作台**：提供文件树、代码查看、终端输出、审批与实时对话

## 4. 技术栈
- **后端**：Go 1.24+、Gorilla WebSocket、Bubble Tea、Connect RPC
- **前端**：React、Vite、TypeScript、Bun
- **外部依赖**：`antigravity_core`、`rg`、`fd`
